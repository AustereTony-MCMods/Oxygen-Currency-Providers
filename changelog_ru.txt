0.10.0b [Все] [2020-01-17]
+ Изменён принцип работы провайдеров, теперь они могут быть использованы для добавления новой виртуальной валюты
или замены существующей. Несколько провайдеров могут работать одновременно. Теперь провайдер требуется и на сервере и на клиенте.
+ Добавлена поддержка кастомизации иконки и названия валюты для виджетов клиента.
+ Добавлены файлы конфигурации, они будут добавлятся в отдельную папку ".minecraft/config/oxygen/currency providers/":
  Опции:
  Сервер
  - "provider_index" - [int] - индекс валюты, который займёт/заменит провайдер
  - "force_balance_sync" - [boolean] - разрешить принудительную синхронизацию баланса. Баланс будет синхронизироваться с клиентом 1  
  раз в секунду. Используйте эту опцию если модификация, для которой используется провайдер, используется как API для других 
  модификаций.
